## 1. 前端构建
```bash
pnpm build
```

## 2. 安装nginx并启动
1. 新建 Nginx 官方 yum 源：
```bash
sudo tee /etc/yum.repos.d/nginx.repo > /dev/null <<'EOF'
[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/7/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key
module_hotfixes=true
EOF
```


2. 安装nginx

```bash
sudo yum clean all
sudo yum makecache
sudo yum install -y nginx
```

3. 立即启动并查看状态
```bash
sudo systemctl enable --now nginx
sudo systemctl status nginx --no-pager
```

## 3. 部署前端到服务器并配置nginx
1. 上传构建产物
```bash
rsync -avz --delete ./dist/ root@host:/tmp/blog-dist
```

2. 将构建文件放到站点目录
```bash
sudo rsync -av --delete /tmp/blog-dist/ /var/www/blog/
sudo find /var/www/blog -type d -exec chmod 755 {} \;
sudo find /var/www/blog -type f -exec chmod 644 {} \;
```

3. 新建nginx站点配置 /etc/nginx/conf.d/blog.conf：
```bash
sudo tee /etc/nginx/conf.d/blog.conf > /dev/null <<'EOF'
  server {
      listen 80;
      server_name wuyin.dev www.wuyin.dev;

      root /var/www/blog;
      index index.html;

      # 前端静态文件 + SPA 路由回退
      location / {
          try_files $uri $uri/ /index.html;
      }

      # 反向代理后端 API（按你的后端端口改）
      location /api/ {
          proxy_pass http://127.0.0.1:49002;
          proxy_http_version 1.1;
          proxy_set_header Host $host;
          proxy_set_header X-Real-IP $remote_addr;
          proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
          proxy_set_header X-Forwarded-Proto $scheme;
      }
  }
  EOF
```

4. 检查并重载nginx
```bash
sudo nginx -t
sudo systemctl reload nginx
```



5. 本机先测连通：
```bash
  curl -I http://127.0.0.1
  curl -I http://yourdomain.com
```


## 4.接 Cloudflare（核心）

  1. Cloudflare 添加站点。
  2. 到域名注册商把 NS 改为 Cloudflare 分配的 2 个 NS。
  3. Cloudflare DNS 里添加：

  - A  @   SERVER_IP（橙云开启 Proxied）
  - A  www SERVER_IP（橙云开启 Proxied）

  4. 等 NS 生效后再继续。

  5. 让用户先过 Cloudflare 验证

  1. Cloudflare Security 开启 Under Attack Mode，会先挑战再放行。
  2. 或在 WAF 自定义规则里对全站设 Managed Challenge。

  6. 防绕过 Cloudflare（非常关键）

  只允许 Cloudflare IP 段访问你服务器 80/443，不允许公网直连源站。
  你可以按 Cloudflare 官方 IP 列表写防火墙白名单（我下一条可以给你 firewalld 版本的完整脚本）。

  7. HTTPS（建议）

  在 Cloudflare 里给源站签发 Origin Certificate，Nginx 配置 443 证书，Cloudflare SSL 模式设为 Full (strict)，并开启 Always Use HTTPS。
